Notes on Isabelle/jEdit Prover IDE

  • Formal Isabelle/Isar text is checked asynchronously while editing.
  • Prover feedback works via tooltips, syntax highlighting, colors, boxes etc. based on semantic markup provided by Isabelle in the background.
  • The modifier key C (CONTROL on Linux or Windows, COMMAND on Mac OS) exposes additional information.
  • Dockable panels (e.g. Output are managed as independent windows by jEdit. Multiple instances are possible.
